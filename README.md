A Next.js project demonstrating an interactive, bento-style UI with fluid animations using [Motion](https://motion.dev/).

## Features

-   **Interactive States**: Seamless transitions between Pill, List, Form, Modal, and Share views.
-   **Fluid Animations**: Powered by `motion/react` for smooth layout changes.
-   **Modern UI**: Clean, glassmorphic design with Tailwind CSS.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open the app**:
    Navigate to [http://localhost:3000](http://localhost:3000).
